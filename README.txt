Mi Proyecto Web de Gestión de Inventarios es una aplicación web desarrollada en Django que simplifica la gestión de inventarios para pequeñas empresas. Con esta aplicación, los usuarios pueden administrar categorías de productos, clientes y productos de manera eficiente. También pueden realizar búsquedas en la base de datos para encontrar información específica rápidamente.

## Instalación

```bash
# Clona el repositorio
git clone https://github.com/tu_usuario/tu_proyecto.git

# Instala las dependencias
pip install -r requirements.txt

# Realiza migraciones
python manage.py migrate

# Inicia el servidor de desarrollo
python manage.py runserver

Uso
Accede a la aplicación en http://localhost:8000
Gestiona categorías, clientes y productos a través de la interfaz web.
